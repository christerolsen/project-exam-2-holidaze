## Description

Holidaze is a venue booking website that allows users to browse, book, manage, and host vacation properties. Users can create and manage venues, view bookings, and manage their profile details. This website is built with React, leveraging a backend API for data management and user authentication.

### Features

- **Venue Browsing**: Users can view a list of venues, search for specific venues, and access detailed venue pages with calendars showing available dates.
- **User Management**: Users with a stud.noroff.no email can register as customers or venue managers, log in, log out, and update their avatars.
- **Customer Features**: Registered customers can create bookings for venues and view their upcoming bookings.
- **Venue Management**: Registered venue managers can create, update, and delete venues they manage, as well as view bookings for their venues.

## Getting Started

### Installing

1. Clone the repo:

```bash
git clone https://github.com/christerolsen/project-exam-2-holidaze.git
```

2. Install the dependencies:

```bash
npm install
```

### Running

To run the app, run the following commands:

```bash
npm run dev
```
